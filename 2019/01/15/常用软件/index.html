常用软件

资源收集

Windows软件

  • Cmder 命令行工具
  • Snipaste 截图
  • Everything 快速查找文件
  • OBS Studio 屏幕录制
  • GifCam gif录制
  • ShareX 截图录屏录制gif
  • KeyCastOW 在屏幕上显示按键操作
  • SpaceSniffer 显示磁盘占用
  • Process Explorer 显示进程占用
  • POTPlayer 视频播放
  • Sonic Visualiser 音频工具
  • Cakewalk 编曲软件
  • Trello 看板工具
  • Eagle 设计师图片管理工具
  • Typora Markdown编辑器 收费了
  • WinMD5Free MD5验证,快速求MD5
  • PanDownload 百度云盘下载工具
  • Acronis True Image 备份还原系统,用这个大概20分钟就把两个盘200G的内容备份完了，然后再30分钟左右还原好,好于dism++备份(群友 纱代推荐)
  • Bongocat 直播录像显示键盘鼠标操作
  • clumsy 能在 Windows 平台下人工造成不稳定的网络状况，方便你调试应用程序在极端网络状况下的表现。

VisualStudio插件

  • Supercharger CodeMap功能(社区版免费,大部分情况需要付费)
  • CodeNav CodeMap功能(免费)
  • File Path On Footer 左下角显示文件路径
  • File Encoding Info 左下角显示文件路径和文件编码格式
  • Generate GUID for Visual Studio 生产多种格式GUID
  • ClaudiaIDE 更换VS背景
  • RoslynClrHeapAllocationAnalyzer 基于Roslyn的C＃堆分配诊断分析器，可以检测显式和许多隐式分配，如装箱，显示类，闭包，隐式委托创建等。
  • Visual Studio IntelliCode 增强智能感知。
  • PowerMode 增加酷炫的打字特效。
  • OzCode 神级调试器
  • Code Metrices 代码复杂度评估插件
